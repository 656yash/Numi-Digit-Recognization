Numi Digit Recognition

A web-based Handwritten Digit Recognition system built using Flask, TensorFlow Lite, and the MNIST dataset. Users can draw a digit (0–9) on an interactive canvas, and the trained Convolutional Neural Network (CNN) predicts the digit in real time with confidence scores.

🚀 Features
✍️ Interactive drawing canvas
🔢 Real-time handwritten digit prediction
🧠 CNN model trained on the MNIST dataset
⚡ Optimized TensorFlow Lite inference
🌐 Lightweight Flask web application
☁️ Deployment-ready for platforms like Vercel
📸 Project Overview

The application allows users to:

Draw a digit on the canvas.
Submit the drawing for prediction.
Receive:
Predicted digit
Model confidence score

The backend preprocesses the image and performs inference using a TensorFlow Lite model for faster execution and lower resource consumption.
